# Caesar 1.0.0

## This is an C++ simple caesar cipher application

## Caesar 1.0.0 [https://github.com/knhn1004/caesar]

### Usage: caesar [Options] {target string} (-k {key})

### Options

- -h: help message
- -e: encrypt string
- -d: decrypt code
- -b: brute force code
- -k: brute force code
- -b: brute force code

### Examples

```bash
caesar -e oliver -k 15
```

```bash
- caesar -d daxktg -k 15
```

```bash
- caesar -b daxhtg
```

```bash
- caesar -i
```
